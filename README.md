# dipadanuwar-portfolio

A simple single-page portfolio built with HTML, CSS, and JavaScript.

## Contents

- `index.html` — main entry  
- `css/` — stylesheets  
- `js/` — JavaScript behavior  
- `img/` — images and media  
- `lib/` — third-party libraries (if present)  
- `.github/` — workflows and repo metadata

## Features

- Single-page layout  
- Responsive (mobile and desktop)  
- Interactive UI powered by vanilla JavaScript  
- All static assets included

## Preview

Open `index.html` in your browser, or use a local development server (for example, the Live Server extension in VS Code).

Live preview: https://dipadanuwr-alt.github.io/dipadanuwar-portfolio/

## Deployment

Deploy as a static site (GitHub Pages, Netlify, Vercel, etc.). For GitHub Pages, enable Pages in the repository settings and point it to the branch you prefer (commonly `main`). The site is published at the preview link above when Pages is enabled.
